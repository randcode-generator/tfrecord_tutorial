# tfrecord tutorial
This is the source code for the [tf record tutorial](https://machine-learning-site.web.app/tensorflow_tfrecord.html)

## How to run
1. Install Pillow

`python -m pip install Pillow`

2. Install lxml

`python -m pip install lxml`

3. Create tfrecord

`python tfrecord_write.py`

4. Read tfrecord

`python tfrecord_read.py`
